# GistPin 🌍📍

**GistPin** is an open, location-aware platform for anonymous, hyperlocal expression.  
It allows people to drop short messages (“gists”) pinned to real-world locations—sharing tips, alerts, stories, or banter visible to others nearby.

GistPin is built for **local communities**, **travelers**, and **urban explorers** who want real-time, place-based insights without identity friction.  
The platform combines Web2 infrastructure with **Web3 smart contracts on the Flare Network** for transparency, permanence, and trust.
